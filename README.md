Simulation Code

These programs are Python script that reproduce from the papers.

1. A Dynamical Model of Subjectivity: Toward an Integrative Computational Architecture for the Mind

DOI: https://doi.org/10.51094/jxiv.1794

2. The Geometry of Neural Scaling: From Empirical Laws to Geometric Saturation

DOI: https://doi.org/10.51094/jxiv.????


> Requirements
Python 3

Libraries listed in requirements.txt


> Installation

Clone this repository or download the source code.

Install the required libraries:

pip3 install -r requirements.txt


> Usage

Run the script from your terminal:

1. A Dynamical Model of Subjectivity: Toward an Integrative Computational Architecture for the Mind

python3 Subjectivity.py

2. The Geometry of Neural Scaling: From Empirical Laws to Geometric Saturation

python3 Geometry.py


> Expected Output:

pdf will be created in the same directory where the script is located.
